# 📊 CRM Pro - PWA Wrapper (محمي بكلمة مرور)

## نظام إدارة العملاء - تطبيق ويب سريع وخفيف ومحمي

---

## 🔐 الحماية

التطبيق محمي بـ **Login Screen** — أي حد يفتح اللينك لازم يدخل:
- **اسم مستخدم**
- **كلمة مرور**

### بيانات الدخول الافتراضية:
| المستخدم | كلمة المرور |
|----------|-------------|
| admin | admin123 |
| user1 | pass123 |
| موظف | 123456 |

> ⚠️ **غيّر البيانات دي قبل ما تبعت للعميل!**

## 📱 كيف يستخدم العميل؟

1. **يفتح اللينك** على الموبايل
2. **يدخل اسم المستخدم وكلمة المرور**
3. **يضغط "تسجيل الدخول"**
4. **يفتح CRM بتاعك!**
5. (اختياري) **يثبت التطبيق على الشاشة**

---

## 🔧 تخصيص بيانات الدخول

افتح `index.html` وغير في الجزء ده:

```javascript
const USERS = {
  "admin": "admin123",     // غيّر ده
  "user1": "pass123",      // غيّر ده
  "موظف": "123456"         // غيّر ده
};
```

أضف مستخدمين جدد:
```javascript
"اسم_جديد": "باسورد_جديد"
```
